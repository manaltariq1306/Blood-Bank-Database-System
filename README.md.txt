# Blood Bank Management System

This project is a Blood Bank Management System developed as part of the  
Third Semester Database Systems course.

The system demonstrates database design, relationships, and SQL queries
through a practical PHPâ€“MySQL based application.

## Project Overview
The system manages blood donors, recipients, hospitals, blood samples,
donations, testing, and transfusions. It allows storing, retrieving,
and reporting data using relational database concepts.

## Features
- Donor management
- Recipient management
- Hospital records
- Blood donation tracking
- Blood sample storage and expiry tracking
- Blood testing and test results
- Blood requests and transfusion records
- Report generation using SQL queries

## Database Design
The database includes the following main tables:
- Donor
- Recipient
- Hospital
- Donation
- BloodSample
- Test
- BloodTestResult
- Request
- BloodTransfusion
- Address

Primary keys, foreign keys, and constraints are used to maintain
data integrity and relationships.

## How to Run the Project
1. Install XAMPP
2. Copy the project folder into the `htdocs` directory
3. Create a database named `bloodbank` in phpMyAdmin
4. Import the SQL file located in the `database` folder
5. Update database credentials in `db.php` if required
6. Open the browser and run:
